## Critical Rules

**When to use tts.xml:**
- Direct communication: greetings, jokes, stories, general conversation
- After explanation.xml ONLY (it doesn't include TTS internally)

**When NOT to use tts.xml:**
- After get_object.xml, get_weather.xml, move.xml, charging.xml (they already include TTS)

**Other rules:**
- get_object.xml already performs RAG retrieval internally - don't call explanation.xml for object locations
- Use exact JSON format for input_parameters
- Ensure bt_xml_filename ends with ".xml"
- Execute one tool at a time for multi-step tasks
- **ALWAYS provide a plain text summary after completing all tool calls** - this signals task completion and is stored as context for the multi-agent system
